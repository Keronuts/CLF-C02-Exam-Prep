2. Video: Machine Learning and Artificial Intelligence in a Nutshell (it_clawsp23_13_enus_02)

After completing this video, you will be able to define machine learning and artificial intelligence.

  • define machine learning and artificial intelligence
[Video description begins] Topic title: Machine Learning and Artificial Intelligence in a Nutshell. Presented by: Michael J. Shannon. [Video description ends]

Machine learning, or ML, assists organizations in generating growth, discovering new revenue streams, and fixing challenging issues. Data is a critical success factor (CSF) in business decision-making, and companies have typically leveraged data from various sources, such as customer feedback, their employees, finance, and external consultants.

Machine learning research automates and optimizes this process. Businesses can get results quicker by using code that analyzes very large data sets at high speeds. Some machine learning use cases are: Manufacturing- to support predictive maintenance, quality control, and inventive research in the manufacturing sector.

ML technology also assists companies in improving logistical solutions, supply chain disruptions, and inventory control such as JIT, Just-in-Time. Healthcare and life sciences- the explosion of wearable sensors has resulted in a substantial amount of health data being generated.

Machine learning solutions can analyze the data and support physicians, and other healthcare professionals in performing real-time diagnosis and treatment. Financial services- projects enhance risk analytics and regulation adherence.

ML technology can help investors recognize new prospects by analyzing stock market trends, appraising hedge funds, or adjusting portfolios. Retail- ML initiatives can improve customer service, stock or inventory management, upselling, and cross-channel marketing.

Media and entertainment- this sector uses ML to better appreciate their target audiences and distribute immersive, tailor-made or bespoke, and on-demand content. ML algorithms are used to help design trailers and other advertisements, offer customers content recommendations, and streamline production.

There's four main machine learning algorithms. Supervised machine learning, is where data scientists offer algorithms with branded and distinct training data to evaluate for correlations. Unsupervised machine learning- this is a technique that trains on unlabeled data by scanning new data and establishing significant links between the inputs and predetermined outputs.

Semi-supervised learning- this combines supervised and unsupervised learning by leveraging a small amount of labeled data and a large amount of unlabeled data to train the ML system. And reinforcement machine learning- this is a technique that associates reward values with the different steps that the algorithm must follow.

Technically speaking, there is a difference between machine learning and artificial intelligence. In a nutshell, AI is a domain of computer science focused on solving cognitive problems universally associated with human intelligence: learning, problem solving, and pattern recognition. Although AI can often invoke robotics or even futuristic dystopian panoramas, it transcends the automatons of science fiction into the non-fiction of modern-day advanced computer science.

AWS offers ready-made intelligence for customer applications and workflows with pre-trained AI services. These AI services effortlessly integrate with applications to support common use cases such as personalized recommendation engines; contact center modernization; safety and security enhancement; and customer engagement improvement.

AWS AI services leverage the same deep learning technology that powers Amazon. Customers attain quality and accuracy from continuously learning application programming interfaces without the need for machine learning experience.

Amazon Athena is a serverless, interactive analytics solution built on open-source frameworks. It supports open-table formats and file formats.

Athena delivers a flexible and easy way to analyze petabytes of data right where it resides. Athena is serverless, highly scalable, and offers streamlined analytics service for all of your data. AWS Glue. AWS Glue is a serverless data integration service that makes it easier to discover, prepare, move, and integrate data from multiple sources.

Glue is used for analytics, machine learning, and application development. AWS customers can use Glue to discover and connect to over 70 varied data sources; manage data in a centralized data catalog; and visually create, run and monitor ETL- extract, transform, and load pipelines, to load data into data lakes.

Some of the Glue data sources include S3, DynamoDB, Amazon Relational Database Service (RDS), database running on EC2 instances, and other databases, and SaaS providers or AWS marketplace providers. [Video description begins] A flowchart containing four boxes is displayed on the screen. The first box is titled Data sources and lists various data sources in it. This is connected u to the second box titled Choice of interfaces which in turn is connected to a third box titled Data integration engines using to and fro arrows. The second box is also connected to a fourth box containing data lakes and data warehouse options. [Video description ends]

Glue's open interfaces support interactive and job workloads. You can choose between different interfaces such as AWS Glue Studio, Amazon SageMaker notebooks, or other notebooks and IDEs, Integrated Development Environments.

Then you can choose a preferred serverless, scalable data processing engine such as AWS Glue for Ray, AWS Glue for Python Shell, or AWS Glue for Apache Spark, with automatic scaling, and pay-as-you-go pricing, ultimately sending the data to Amazon Redshift data warehousing, S3 data lakes, or other data warehouses.

If you’re going to use Amazon S3 with AWS Glue, a popular solution, simply load the new data into an S3 bucket, then, initiate one or more ETL jobs. [Video description begins] A flowchart labelled AWS Glue is displayed on the screen. It contains 6 boxes and the first box titled Amazon S3 is connected to another titled AWS Glue. [Video description ends] You can use the AWS Glue Data Catalog to manage data, with the data catalog serving as a central metadata repository.

You can use AWS Glue ETL to schedule the ETL jobs to transform and load the data. You can monitor logs and push notifications to Amazon CloudWatch, and then ultimately load your transformed data into your targeted data store- an Amazon Redshift cluster, or an Amazon S3 bucket.
